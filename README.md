# Project Overview

This project is a Rock Paper Scissors game bot built using Python.

# How the Bot Works (Simple Explanation)
The bot remembers the opponent‚Äôs previous moves.

It checks which move the opponent uses the most.

It plays the winning move against that choice.

If no data is available, it plays randomly.

Example:
If opponent often plays Rock, the bot will play Paper.
This is a basic form of pattern recognition, which is a key idea in Machine Learning
